# Adam Schindler Plumbing & Heating Website

A modern, responsive React website for Adam Schindler Plumbing & Heating, showcasing their commercial and industrial plumbing services in Massachusetts.

## Features

- **Hero Carousel**: Auto-rotating image carousel showcasing key services and company trucks
- **Services Section**: Four main service offerings with detailed descriptions
  - Installation/Maintenance of Plumbing Systems
  - 24/7 Emergency Services
  - Process Piping
  - Heating Systems
- **Experience Stats**: Key company highlights (15+ years, contracts, certifications)
- **About Section**: Company philosophy and approach
- **Service Areas**: Geographic coverage information
- **Client Testimonials**: Carousel of customer reviews
- **Contact Section**: Multiple contact methods displayed
- **Fully Responsive**: Mobile, tablet, and desktop optimized

## Tech Stack

- **React 18.2**: Modern React with hooks
- **Tailwind CSS 3.3**: Utility-first CSS framework
- **React Icons**: Icon library for UI elements
- **HTML5**: Semantic markup
- **CSS3**: Custom animations and transitions

## Design Highlights

- Blue color theme matching Schindler Plumbing branding
- Clean, modern layout inspired by Amer Plumb design
- Smooth animations and transitions
- Professional typography with Inter font family
- Optimized images from Schindler Plumbing website

## Installation

```bash
# Install dependencies
yarn install

# Start development server
yarn start

# Build for production
yarn build
```
